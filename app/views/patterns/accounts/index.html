Usage

Account patterns should be used when there is a need to display the details of a specific user, retailer or manufacturer account. By following these account patterns, all DVLA services will have common looking accounts.

Header

All account pages should begin with a header that clearly and succinctly describes that account. When using account headers, it is suggested that you include the following key bits of information:

  1. An icon that indicates the account type
  2. A label that indicates the account type
  3. A label that indicates the account ID
  4. A large heading that displays the account name
Account headers have been built to be flexible, allowing developers to alter these fields if some are more or less flexible. Here are some examples of account headings for a retailer, manufacturer and user.
