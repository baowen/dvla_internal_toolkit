Prerequisites

In order to use the prototype kit, you must have installed the following on your development machine

  • Ruby & Ruby on Rails
  • Text editor of your choice

Installation

Ruby

After install Ruby and Ruby on Rails on your device, you are ready to create your prototype. To do so, run the following command

rails new my-prototype

Rails will have now generated a new project named my-prototype for you. Now, in an editor of your choice, open your newly created project then open the Gemfile and add the following line.

gem 'dvla_internal_frontend_toolkit', '~> 0.3.2'

This will add the DVLA frontend kit as a depenancy to your prototype, which in turn will give you access to all of the internal elements and styles. Finally, now that you've added the dependancy to the project, you have to tell your project to update it's depenancies. To do this, run the following command

bundle update

You are now ready to begin working on your prototype.

Running

Ruby

Now that your Rails project has all it's depenancies installed, you're ready to run the project. To do so, run the following command

rails server

This will start your prototype app running on localhost:3000.
