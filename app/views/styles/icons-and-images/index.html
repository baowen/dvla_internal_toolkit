Using the font

DVLA icons are provided as the font 'dvla-icons'. All icons are accessible through the use of classes or unicode values. Classes are used by combining the icon class along with the corresponding class for that icon. For example:

To access the font directly with unicode, use font-family: 'dvla-icons'; or font-family: $icon-font; and set the corresponding content: ''; value in the :before or :after of your element.
